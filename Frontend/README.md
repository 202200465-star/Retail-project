# 🎨 Frontend — ShopSphere E-Commerce Platform

## 📌 Overview

This folder contains the frontend application of the ShopSphere retail e-commerce platform.

The frontend delivers a modern, responsive, and interactive shopping experience using Vue.js and Bootstrap.

---

# 🚀 Features

- Modern Landing Page
- Product Catalog
- Search & Filtering
- Shopping Cart Sidebar
- Wishlist System
- FAQ / Help Center
- User Authentication
- Profile Management
- Order History Tracking
- Responsive Design
- Admin Dashboard Interface

---

# 🛠️ Frontend Tech Stack

- Vue.js 3
- Vue Router
- Bootstrap 5
- Bootstrap Icons
- Axios
- Vite

---
