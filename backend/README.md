# Backend — Retail E-Commerce Web Application

This folder contains the **Backend** (server-side) of the EC385 retail e-commerce project (B2C model).
The backend provides REST APIs for authentication, product management, cart/checkout, and orders.

---

## Tech Stack (Planned)
- Runtime: Node.js
- Framework: Express.js
- Database: (TBD) MongoDB / MySQL / PostgreSQL
- Authentication: JWT (planned)
- API Style: REST
