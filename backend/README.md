# ⚙️ Backend — ShopSphere E-Commerce Platform

## 📌 Overview

This folder contains the backend server and REST API for the ShopSphere e-commerce platform.

The backend handles:
- Authentication
- Product Management
- Orders
- Cart Operations
- Admin Dashboard Logic
- Database Communication

---

# 🚀 Backend Features

- JWT Authentication
- Role-Based Authorization
- Product CRUD Operations
- Order Management System
- Order Status Tracking
- User Management
- RESTful API Architecture
- MVC Pattern
- Middleware-Based Security

---
